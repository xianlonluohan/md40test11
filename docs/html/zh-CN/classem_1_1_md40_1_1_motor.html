◆ PhaseRelation

enum class em::Md40::Motor::PhaseRelation : uint8_t
strong

用于明确电机正转时编码器AB相的相位关系，以便在脉冲计数及后续速度计算等操作中依据正确的相位关系进行处理。

枚举值
kAPhaseLeads 0 

表示电机正转时A相领先于B相。

kBPhaseLeads 1 

表示电机正转时B相领先于A相。

◆ State

enum class em::Md40::Motor::State : uint8_t
strong

电机状态枚举。

枚举值
kIdle 0 

表示电机处于空闲状态。

kRuningWithPwmDuty 1 

表示电机处于PWM占空比模式运行。

kRuningWithSpeed 2 

表示电机处于速度模式运行。

kRuningToPosition 3 

表示电机正在执行位置闭环运动，向目标位置运行中。

kReachedPosition 4 

表示电机到达目标位置状态。
